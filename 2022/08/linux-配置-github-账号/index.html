Step 2 配置 SSH 公钥

登录 GitHub 主页，在个人设置中选择“SSH and GPG keys”，单击“New SSH key”，将上一步中生成的公钥（id_ecdsa.pub）复制进来（下图）。

2

[kwang@Slave02 ~/Prometheus]$ git config --global user.email "prc.wkai@gmail.com"
